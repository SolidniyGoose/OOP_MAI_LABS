# Лабораторная 1 (Вариант 6):

Написать функцию, которая принимает строку и возвращает новую строку с удалёнными гласными.
Например, строка "This website for losers LOL!" превратится в "Ths wbst s fr lsrs LL!"

# Лабораторная 2 (Вариант 3):

рой. Младшая цифра имеет меньший индекс (единицы — в нулевом элементе массива).
