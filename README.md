# ООП - ЛАБОРАТОРНЫЕ РАБОТЫ

# Лабораторная 2 (Вариант 3):
```
Создать класс Octal для работы с беззнаковыми целыми восьмеричными числами, используя для представления числа массив из элементов типа unsigned char, каждый элемент которого является восьмеричной циф
рой. Младшая цифра имеет меньший индекс (единицы — в нулевом элементе массива).
```

# Сборка, запуск и тесты:
```
mkdir build && cd build
cmake ..
make

./main
./tests
```
